## 📋 작업 배경

계산기의 버튼 패드를 구현합니다.

## 🎯 작업 내용

- 숫자 버튼 (0-9)
- 연산자 버튼 (+, -, *, /, 괄호, 소수점)
- 함수 버튼 (sin, cos, tan, log, ln, ^, sqrt)
- 특수 버튼 (C, DEL, =)

## ✅ 인수 조건 (Acceptance Criteria)

- 모든 버튼 클릭 시 정상 동작
- 버튼 호버 효과 적용
- 터치 친화적 크기 (최소 44px)
